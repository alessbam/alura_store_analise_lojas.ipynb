Alura Store – Análise de Desempenho das Lojas
Visão Geral do Projeto

Este projeto tem como objetivo analisar o desempenho de quatro lojas pertencentes ao e-commerce Alura Store. A análise foi realizada para ajudar o proprietário do negócio a decidir qual loja deve ser vendida para obter capital e investir em um novo empreendimento.

Utilizando Python e técnicas de análise de dados, foram avaliados diferentes indicadores de desempenho para comparar as lojas e apoiar uma decisão estratégica baseada em dados.

Problema de Negócio

O Sr. João possui quatro lojas online dentro da Alura Store e deseja vender uma delas para investir em um novo negócio.

Para auxiliar nessa decisão, foi realizada uma análise exploratória de dados com o objetivo de identificar qual loja apresenta o menor desempenho em comparação com as demais.

Métricas Analisadas

Durante a análise, foram avaliados os seguintes indicadores:

Faturamento total por loja

Categorias de produtos mais populares

Média de avaliação dos clientes

Produtos mais e menos vendidos

Custo médio de frete

Essas métricas permitem avaliar tanto o desempenho comercial quanto a satisfação dos clientes.

Ferramentas Utilizadas

A análise foi realizada utilizando as seguintes tecnologias:

Python

Google Colab

Pandas – manipulação e análise de dados

Matplotlib – criação de gráficos e visualizações

Fonte dos Dados

Os dados utilizados neste projeto foram fornecidos pela Alura, como parte de um desafio prático da formação em Data Science do programa Oracle Next Education (ONE).

Cada conjunto de dados representa as vendas de uma loja da Alura Store.

Principais Insights

A análise dos dados revelou que:

Existe diferença significativa no faturamento total entre as lojas.

Algumas lojas apresentam melhores avaliações médias dos clientes.

O custo médio de frete varia entre as lojas, podendo impactar a decisão de compra.

A popularidade das categorias de produtos também varia entre as lojas.

Recomendação Final

Com base nos dados analisados, foi possível identificar que:

A Loja 4 apresenta o menor faturamento total entre as quatro lojas.

Apesar de possuir um custo médio de frete competitivo, o desempenho financeiro da Loja 4 é inferior ao das demais.

Dessa forma, a recomendação baseada na análise é que o Sr. João venda a Loja 4, mantendo as lojas com melhor desempenho e utilizando o capital obtido para investir em um novo negócio.

Visualizações

O projeto inclui gráficos que ajudam a compreender melhor o desempenho das lojas, como:

Comparação de faturamento entre as lojas

Comparação da média de avaliação dos clientes

Distribuição do faturamento entre as lojas

Essas visualizações auxiliam na interpretação dos dados e na tomada de decisão.
